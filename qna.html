Question-1: What are the different ways to select an element in the DOM?

Ans: There are multiple ways to select elements in the DOM. Here are some most commonly used methods:

  • 1. getElementById()
  • 2. getElementsByClassName()
  • 3. querySelector()
  • 4. querySelectorAll()
However, each of these methods can be used depending on the specific use case in your JavaScript code.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: In JavaScript, the properties are used to get or set the content of an HTML element.

  • 1. innerHTML: sets HTML content, including tags
  • 2. innerText: sets visible text, considering styles (like display: none).
  • 3. textContent: Gets or sets all text, ignoring tags and styles.
However, each of these methods can be used depending on the specific use case in your JavaScript code.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique in JavaScript where instead of attaching event listeners to each individual element, you attach a single event listener to a parent element. However, each of these properties can be used depending on the specific use case in your JavaScript code.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a concept in JavaScript where an event triggered on an element propagates upwards through its ancestors in the DOM tree, starting from the target element and moving to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: To create, add, and remove elements using JavaScript I can use the following methods:

  • 1. Create an Element : Use document.createElement() to create a new element.
  • 2. Add an Element : To add the newly created element to the DOM, use appendChild() or insertBefore().
  • 3. Remove an Element : Use removeChild() to remove an element from its parent.
